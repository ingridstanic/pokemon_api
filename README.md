# Who's that Pokémon?

A fun and interactive web application where players test their Pokemon knowledge by guessing random Pokemon based on their appearance. Fetch data in real-time from the PokemonAPI and challenge yourself to guess correctly within 3 attempts!

## Features

- **Random Pokemon Generation**: Each game loads a random Pokemon from the PokemonAPI
- **Interactive Guessing Game**: Players get 3 attempts to guess the correct Pokemon name
- **Visual Feedback**: Pokéball indicators show remaining attempts
- **Responsive Design**: Styled with SCSS for a polished user interface
- **Real-time API Integration**: Fetches live Pokemon data from the official PokemonAPI

## Tools & Technologies

- **TypeScript**: Type-safe development for robust code
- **Vite**: Fast build tool and development server
- **SCSS**: Advanced styling with variables and modularity
- **PokemonAPI**: Official Pokemon data source
- **HTML5**: Semantic markup structure
